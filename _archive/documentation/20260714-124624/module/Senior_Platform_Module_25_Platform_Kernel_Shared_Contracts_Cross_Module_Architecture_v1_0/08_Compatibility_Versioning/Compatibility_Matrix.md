# Compatibility Matrix

## Baseline
Module 25 v1.0 is compatible with Modules 01–24 v1.0.

## Required Compatibility Checks
- Event envelope compatibility
- Configuration envelope compatibility
- Identifier reference compatibility
- Tenant boundary compatibility
- Permission naming compatibility
- Error envelope compatibility
- Audit envelope compatibility
- API metadata compatibility
- Dependency direction compatibility

## Failure Handling
Compatibility failures must be classified as:
- advisory
- blocking
- migration_required
- security_blocking
- tenant_boundary_blocking
