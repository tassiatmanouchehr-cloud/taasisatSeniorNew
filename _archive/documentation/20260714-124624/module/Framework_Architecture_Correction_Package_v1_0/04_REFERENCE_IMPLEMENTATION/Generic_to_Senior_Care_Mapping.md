# Generic to Senior-Care Reference Mapping

This mapping is non-normative. It exists only to help implement the framework in a senior-care marketplace.

| Generic Core Term | Senior-Care Reference Term |
|---|---|
| Platform Owner | Platform Owner / platform owner |
| Platform Team Member | Platform support, marketing, accounting, security, IT |
| Organization | Care provider company |
| Organization Staff | Company operators/admins |
| Independent Provider | Independent provider/nurse |
| Organization Provider | Company-affiliated provider/nurse |
| Customer | Patient or patient family |
| Service Recipient | Patient/elder receiving service |
| Trusted Person | Order-scoped trusted family/contact |
| Service Request | Care request |
| Service Execution | Visit/care delivery |
