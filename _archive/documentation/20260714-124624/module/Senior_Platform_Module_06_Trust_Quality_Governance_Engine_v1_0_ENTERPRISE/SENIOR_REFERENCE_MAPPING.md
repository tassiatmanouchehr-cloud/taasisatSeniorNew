# Generic Service Marketplace Reference Mapping

Generic Marketplace Framework remains domain-neutral.

## Mapping

- Customer → Customer or Customer Delegate
- Provider → Independent Provider / Organization Provider
- Organization → Organization
- Platform Owner → Platform Owner
- Booking → سفارش خدمت
- Service → خدمت مراقبتی / Providerی
- Complaint → شکایت Customer، خانواده، Provider، Organization یا Platform Owner
- Dispute → اختلاف رسمی درباره کیفیت، پول، حضور، خسارت یا رفتار
- Compliance → مدارک، مجوزها، صلاحیت حرفه‌ای
- Risk → ریسک تقلب، لغو مکرر، شکایت مکرر، رفتار مشکوک

## Generic Service Marketplace Rule

Company-level roles only access their own organization data.  
Platform-level roles controlled by Platform Owner can access the full marketplace.
